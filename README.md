# E-Commerce Customer Churn Analysis Dashboard

**Interactive Excel Dashboard** analyzing customer churn patterns in an e-commerce dataset using **Pivot Tables, Charts, and Slicers** – built to showcase **data cleaning, analysis, and visualization skills** for Data Analyst roles.

*Full interactive dashboard with slicers for Churn, Complaint, Marital Status, Satisfaction Score & Warehouse*

---

### Project Highlights
- Overall **Churn Rate: 20%**
- Customers with **missing tenure** → **30% churn** (vs 15.6% when known)
- **Satisfaction Score 4** has the **highest churn at 40%**
- **Mobile & Accessory** buyers churn the most at **27%**
- Churned customers got **₹181.64 average less cashback**
- Weak correlation between `DaySinceLastOrder` and `WarehouseToHome`
